# Template — Screener de recrutamento

> Triagem que separa quem entra no estudo. Usar para recrutar participante certo · NÃO usar perguntando autoavaliação ("você é iniciante?") nem intenção futura. Dono → CORE `11`.

## Como usar

Preencha ao recrutar. Pergunte **comportamento passado concreto** ("há quanto tempo usa/comprou X?"), nunca autoavaliação nem "você compraria?". **Não entregue o perfil desejado** na pergunta — senão a pessoa responde o que acha que você quer. Erro comum: screener tendencioso que contamina a amostra antes do estudo começar.

---

```
SCREENER

Objetivo do recrutamento (qual perfil): [ ___ ]

— Inclusão (precisa ter): [ ___ ]
— Exclusão (não pode ter): [ ex.: trabalha com UX/concorrente; já participou < 60 dias ___ ]

Perguntas (comportamento passado, não opinião):
1. [ "Quando foi a última vez que você ___?" → opções que não revelam a resposta certa ]
2. [ "Com que frequência você ___?" → ___ ]
3. [ "Conte do último ___ que você fez." (aberta, confirma experiência real) ]

Pergunta-armadilha (filtra resposta automática / profissional): [ ___ ]
Cotas (quantos por perfil/segmento): [ ___ ]

Disponibilidade (datas/horários): [ ___ ]
Necessidade de acomodação / acessibilidade: [ ___ ]
Consentimento para contato: [ ] sim  [ ] não
```

→ Entenda o método: CORE `11`. Cuidado com viés de recrutamento → `03`.
