# Checklist — Síntese

> Qualidade da síntese antes de entregar. Dono → CORE `04`.

```
- [ ] Os insights têm evidência rastreável até o dado bruto?
- [ ] Os padrões aparecem em mais de uma fonte?
- [ ] As contradições foram registradas (não escondidas)?
- [ ] O escopo do mapa está claro?
- [ ] A jornada tem pessoa, cenário e objetivo?
- [ ] Cada mapa cobre um público por vez?
- [ ] As oportunidades são acionáveis (não vagas)?
- [ ] As recomendações estão ligadas a decisões?
- [ ] As emoções, dúvidas e contexto aparecem (não só ações)?
- [ ] O time sabe o que fazer depois?
```

→ Entenda o método: CORE `04`.
