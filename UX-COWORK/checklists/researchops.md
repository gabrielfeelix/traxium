# Checklist — ResearchOps mínimo viável

> Para time pequeno ou sem cultura de pesquisa. Comece pelo maior gargalo, não monte tudo. Dono → CORE `11`.

```
- [ ] Consentimento padrão pronto (1º passo — destrava ética).
- [ ] Templates básicos: plano de pesquisa, roteiro, screener.
- [ ] Repositório organizado + taxonomia simples (poucas tags).
- [ ] Planilha de participantes com consentimento e regra de retenção.
- [ ] Intake mínimo (qual decisão? qual risco?).
- [ ] Playback curto recorrente + registro de decisões.
```

→ Entenda o método: CORE `11`. Sem cultura de pesquisa / output > validação → `14`. Templates → `templates/`.
