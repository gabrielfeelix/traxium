# Checklist — Plano de pesquisa pronto?

> Antes de iniciar a coleta, confirme que o plano se sustenta. Dono → CORE `01` / `11`.

```
- [ ] Qual decisão esta pesquisa apoia? (sem decisão, não pesquise)
- [ ] Qual risco estou reduzindo (valor/usabilidade/navegação/adoção)?
- [ ] O que já sei (analytics, suporte, pesquisa anterior) e já consultei?
- [ ] O que ainda é suposição que preciso testar?
- [ ] Comportamento ou percepção? Profundidade ou escala?
- [ ] O método cabe no prazo/orçamento e é o mais forte viável?
- [ ] Perfil certo definido e plano de recrutamento (screener)?
- [ ] Consentimento preparado?
- [ ] Que entregável sai disto e quem vai usá-lo para decidir?
- [ ] Como vou documentar a evidência?
```

→ Entenda o método: CORE `01` (escolha) e `11` (operação). Pick rápido → `_METHOD_PICKER`. Template → `templates/plano-de-pesquisa.md`.
